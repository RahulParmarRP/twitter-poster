Tools & Projects

📁 Offline Album Organizer

Browser-based photo organization tool. Organize your photos automatically by date into monthly or daily folders.

Features:

  • Works completely offline in your browser
  • Organize by monthly (2025-08) or daily (2025-08-30) albums
  • Supports multiple image formats (JPG, PNG, GIF, HEIC, RAW, TIFF)
  • Preview organization before applying changes
  • File tree browser with folder navigation
  • Duplicate filename handling

Requirements: Chrome or Edge browser (uses File System Access API)
